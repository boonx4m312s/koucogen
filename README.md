# 勾殳玄の定理の計算
レスポンシブ対応のWeb計算サービスです。 
スマートフォンでも操作可能ですが、図が見切れるのでタブレット以上のモニタを推奨しています。  
見切れた場合はピンチで移動してください。  

# 使い方
勾・殳・玄・勾配の何れか2項目を入力し計算を押下する。  
表の項目に計算結果が反映されます。
